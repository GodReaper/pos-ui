## Project Brief

- **Project**: POS Web - Biller & Admin Console
- **Goal**: Provide a fast, touch-friendly web POS interface for billers and an admin console for managing areas, tables, users, and menu.
- **Current Focus**: Implement the main biller table canvas screen with live table status and inline order view.
- **Primary Personas**:
  - **Biller**: Front-of-house staff managing tables, orders, and bills.
  - **Admin**: Back-office user configuring areas, tables, assignments, and menu.
