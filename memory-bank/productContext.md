## Product Context

- **Problem**: Billers need a clear, real-time view of dining areas and tables to manage occupancy, take orders, and settle bills quickly.
- **Solution**: A responsive web UI that mirrors native POS behavior with table canvases, area selectors, and inline order panels.
- **UX Priorities**:
  - Dark, low-glare theme suitable for dim environments.
  - High-contrast status colors and clear typography.
  - Minimal modals; prefer inline panels and non-blocking loaders.
  - Fast perceived performance with regular polling and subtle state indicators.
